# HSE-ML-Course

## Запуск приложения

```
pip install -r requirements.txt
python webapp/app.py
```

## Выводы
### Что было сделано

#### EDA
- Обработка пропусков
- Анализ дубликатов
- Профилирование данных с помощью ydata-profiling
- Построены распределения числовых и категориальных признаков
- Исследование корреляция между признаками
- Кодирование категориальных признаков

#### Обучение моделей регрессии
- Linear Regression
- Lasso
- Ridge

#### Подборд параметров
- KFold
- GridSearch

#### Бизнес метрика
- Выбор лучшей модели по кастомной бизнес метрике

#### Приложение
- EDA по данным
- Интерфейс предсказания по файлу/единичное предсказание
- Анализ модели

### Результаты
Обучение модели показало, что Scaler не даёт большого прироста, в то время как использование всех признаков (и категориальных и вещественных) дает сильный прирост

#### Что получилось хорошо
- Качественная предобработка
- Подбор гиперпараметров

#### Что получилось плохо
- Большая дисперсия ошибок для автомобилей >3М из-за малых данных
- Не самое лучшее качество модели
